# One Page Website - Lightbox Gallery

## 📌 Overview
A simple one-page website featuring a responsive image gallery with a lightbox (modal) effect built using HTML, CSS, and JavaScript.

## 🚀 Features
- Image thumbnails
- Lightbox modal view
- Next/Previous navigation
- Close button
- Responsive design

## 🛠️ Technologies
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Download or clone the repo
2. Open `one_page_website.html` in your browser

## 📁 Structure
- `one_page_website.html`
- `css/styles.css`
- `js/one_page_website.js`
- `images/`
